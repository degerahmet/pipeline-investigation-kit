# Pipeline Investigation Kit

A lightweight, serverless investigation toolkit for debugging data pipeline issues:
**delays**, **gaps**, **duplicates**, and **replay**.

- Ingest raw events (immutable to S3)
- Index metadata in DynamoDB
- Replay by entity + time window to SQS
- Optional processor to compute placeholder aggregates
